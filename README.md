# 文章添加指南

### 在_posts 文件夹下存放markdown格式的文章

### 在文章最前面需要添加文章头

```
---
layout: post
title: 文章标题
date:   时间  
cover: 封面
categories: 类别
tags: 标签
---
```

时间格式：2018-02-09

封面为图片的url 可用工具生成 

类别和标签填一样的内容 

有： | 最新动态 | 电子爱好者 | 程序猿 | 资源分享 |
